An open-source tool to explore, analyze and manipulate GeoJSON data using JavaScript.
local_library Use common web mapping technologies
travel_explore Explore and transform geospatial data
science Prototype and experiment with geospatial scripting

Features

  • Add colour-coded GeoJSON data layers
  • Use data layers within the script without affecting the original data
  • Return GeoJSON data from the script to render it in the map
  • Download any GeoJSON data layer to a local file
  • Download the script to a local file
  • Click on any feature in the map to view its GeoJSON data
  • Double-click on the map to quickly create a single point layer
  • Resize panes to suit a given workflow
